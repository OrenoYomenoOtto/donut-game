# donut-game

## 1.概要

ドーナツのパズルゲームを作る!!

## 2.環境

- 日本語
- windows10
- [dotnet](https://dotnet.microsoft.com/ja-jp/) 9.0.102
- [Raylib-cs](https://github.com/raylib-cs/raylib-cs) 7.0.1

## 3.インストール方法

1. このリポジトリをクローンします：<https://github.com/OrenoYomenoOtto/donut-game.git>
2. Raylibをインストールします、ターミナルで次を実行してください： `dotnet add package Raylib-cs`
3. プロジェクトをビルドして実行します: `dotnet run`
